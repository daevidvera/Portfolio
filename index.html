Minecraft Snake Game
ECE 2035 Project

I developed a fully playable Snake game on an MBed microcontroller, using C++ and a custom Doubly Linked List (DLL) to manage dynamic memory in real-time. The game runs on a uLCD screen with input from a navigation switch, supporting features like item spawning, collision detection, boost mechanics, and optimized growth logic. This project highlights my low-level programming skills, real-time embedded development, and game logic design.
